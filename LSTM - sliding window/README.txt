This folder contains the implementation of LSTM for time-series regression in the northern Adriatic Sea, considering the sliding window approach, specifically implemented for LSTM.

To run the ML emulator the run_training.py script must be launched. This code reads the config.yaml file ('configs' folder), in which the following can be defined: output directory, model to use (the definition of the models can be found in models.py ('src' folder)), model configuration parameters, loss function, among other.
